# Dev Toolbox（Chrome 插件）

日常开发工具箱（Manifest V3）：时间戳转换、JSON 格式化、文本/列表互转、文本转义。

## 安装与发布

### 本地开发安装
1. 安装依赖：`npm i`
2. 构建：`npm run build`
3. 打开 `chrome://extensions` → 开启“开发者模式” → “加载已解压的扩展程序” → 选择项目的 `dist/` 目录

### 发布 / 离线安装
详见 [PUBLISHING.md](PUBLISHING.md)（包含发布到 Chrome 商店及打包 `.crx` 的说明）。

## 功能

- **时间戳**：秒/毫秒自动识别；本地/UTC；一键复制输出
- **JSON**：格式化/压缩/校验（显示行列）；可选排序 key；一键复制
- **文本/列表**：自动解析（JSON 数组/多行/分隔符）；trim/去空/去重；多种输出格式
- **文本转义**：转义/去转义；支持换行、制表符、引号等自定义选项

## 开发说明

- `npm run dev`：启动 Vite 预览 UI（不等同于扩展模式）
- 扩展调试推荐：修改代码后 `npm run build`，然后在扩展管理页点击“重新加载”
 - 图标在构建时自动生成（占位用 1x1 PNG），可替换为你自己的 PNG 文件
